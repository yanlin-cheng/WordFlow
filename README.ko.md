# WordFlow v2.0

**WordFlow** 는 로컬 ASR 음성 인식을 지원하는 스마트 음성 입력 도구입니다. 음성을 실시간으로 텍스트로 변환하여 모든 애플리케이션에 입력할 수 있습니다.

**v2.0 완전 리뉴얼** - .NET 8 WPF 로 재구축되어 더욱 안정적이고 지능적인 음성 입력 경험을 제공합니다.

## 🚀 빠른 시작

### 설치 단계

1. **설치 프로그램 다운로드**: `WordFlow_Setup.exe` 다운로드
2. **설치 프로그램 실행**: 패키지를 더블클릭하고 안내에 따라 설치
   - .NET 8 런타임 자동 감지 및 설치
   - 음성 인식 모델 다운로드 및 설치 선택 (권장)
3. **앱 시작**: 설치 완료 후 즉시 사용 가능

### 모델 설치

**권장**: 설치 중 모델 다운로드 옵션을 체크하면 설치 프로그램이 자동으로 모델을 다운로드하고 설치합니다.

**대체 방법**: 설치 중 모델이 다운로드되지 않은 경우:
1. WordFlow 시작
2. "모델 관리자" 버튼 클릭
3. 원하는 모델의 "다운로드" 버튼 클릭
4. 다운로드 완료 후 자동으로 모델 로드 시도

### 사용 방법

1. **WordFlow 시작** - 앱은 시스템 트레이로 최소화됩니다
2. **눌러서 말하기** - 단축키 (기본값: CapsLock) 를 눌러 녹음 시작
3. **놓아서 입력** - 단축키를 놓으면 자동으로 텍스트를 인식하여 입력

## ✨ 기능

### 핵심 기능

| 기능 | 설명 |
|---------|-------------|
| 🎤 **스마트 음성 입력** | 눌러서 말하고, 놓으면 인식 및 입력 |
| 🧠 **로컬 ASR 인식** | 알리바바 DAMO 아카데미의 Paraformer 모델 사용, 오프라인에서 사용 가능 |
| 📚 **개인 어휘** | 사용자 정의 어휘 지원, AI 스마트 수정 |
| 📝 **기록** | 입력 기록 자동 저장, 나중에 참조 가능 |
| 🔔 **트레이 통합** | 시스템 트레이로 최소화, 언제든지 액세스 가능 |
| ⚡ **자동 시작** | 부팅 시 자동 시작 지원 |
| ⌨️ **글로벌 단축키** | 사용자 정의 가능한 단축키 |
| 🔄 **자동 모델 로드** | 다운로드 후 자동으로 모델 로드 시도 |

### 단축키 설정

WordFlow 는 여러 단축키 구성을 지원합니다:

| 단축키 유형 | 기본값 | 설명 |
|-------------|---------|-------------|
| 음성 입력 단축키 | CapsLock | 눌러서 말하고, 놓으면 인식 |
| 사용자 정의 단축키 | 오른쪽 Alt | 설정에서 구성 가능 |

> **💡 팁**: 음성 입력 단축키로 CapsLock 을 사용하는 것을 권장합니다:
> - 중앙에 위치하여 양손으로 쉽게 조작 가능
> - 키 표면이 커서 실수 접촉이 적음
> - 누를 때 명확한 물리적 피드백 제공

### 모델 관리

v2.0 은 모델 관리 기능을 추가했습니다:
- **원클릭 다운로드**: "다운로드" 버튼을 직접 클릭, 반복 선택 불필요
- **자동 로드**: 다운로드 완료 후 자동으로 모델 로드 시도
- **서비스 감지**: ASR 서비스가 실행되지 않은 경우 사용자에게 알림
- **상태 표시**: 실시간 모델 상태 (설치됨/다운로드 안됨/로드됨)

## ❓ 자주 묻는 질문

### Q1: 첫 실행 시 무엇이 다운로드되나요?

첫 실행 시 음성 모델 파일 (약 206MB) 을 자동으로 다운로드합니다. 네트워크에 연결된 상태로 유지하세요. 다운로드 후에는 다시 다운로드하지 않고도 정상적으로 사용할 수 있습니다.

### Q2: 모델 다운로드가 실패하면 어떻게 하나요?

자동 다운로드가 실패한 경우:
1. 네트워크 연결이 정상인지 확인
2. 애플리케이션 재시작
3. 모델 파일을 수동으로 다운로드 (위의 "옵션 2" 참조)
4. 모델 관리자 페이지에서 "새로고침" 버튼 클릭

### Q3: 모델을 다운로드한 후 왜 사용할 수 없나요?

v2.0 은 이 문제를 최적화했습니다:
- 다운로드 완료 후 자동으로 모델 로드 시도
- ASR 서비스가 실행되지 않은 경우 사용자에게 알림
- 그래도 작동하지 않으면 앱을 재시작하거나 "서비스에 연결" 버튼을 수동으로 클릭

### Q4: 인식이 부정확하면 어떻게 하나요?

1. 마이크가 정상적으로 작동하는지 확인
2. 조용한 환경에서 사용
3. 명확하게 발음하고 적당한 속도로 말하기
4. 개인 어휘에 자주 사용하는 어휘 추가

### Q5: 왜 일부 애플리케이션에서 텍스트를 입력할 수 없나요?

일부 애플리케이션은 외부 입력을 제한할 수 있습니다. 다음을 시도하세요:
1. WordFlow 를 관리자로 실행
2. 대상 애플리케이션에서 텍스트 상자를 먼저 클릭하여 포커스 얻기
3. 보안 소프트웨어가 차단하지 않는지 확인

### Q6: 단축키를 어떻게 변경하나요?

1. 시스템 트레이 아이콘 우클릭
2. "설정" 선택
3. "단축키 설정"에서 변경

### Q7: 부팅 시 자동 시작되지 않는 이유는 무엇인가요?

1. "부팅 시 자동 시작"이 체크되어 있는지 확인
2. 보안 소프트웨어가 자동 시작을 차단하지 않는지 확인
3. 작업 관리자의 "시작 프로그램" 탭에서 확인
