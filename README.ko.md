# WordFlow v1.0

WordFlow 는 로컬 ASR 음성 인식을 지원하는 지능형 음성 입력 도구입니다. 음성을 실시간으로 텍스트로 변환하여 임의의 애플리케이션에 입력할 수 있습니다.

**v1.0 릴리스** - .NET 8 WPF 로 구축되어 안정적이고 지능적인 음성 입력 경험을 제공합니다.

## 🚀 빠른 시작

### 설치 단계

1. **설치 프로그램 다운로드**: `WordFlow_Setup.exe` 다운로드
2. **설치 프로그램 실행**: 더블클릭하고 지침에 따름
3. **애플리케이션 시작**: 설치 후 WordFlow 시작
4. **모델 다운로드**: 첫 번째 실행 시 모델 다운로드가 자동으로 시작됩니다. 또는 '모델 관리자' → '다운로드' 클릭

### 사용 방법

1. **WordFlow 시작** - 애플리케이션은 시스템 트레이로 최소화됩니다
2. **눌러서 말하기** - 음성 입력 단축키를 눌러 녹음 시작
3. **놓아서 입력** - 단축키를 놓아 인식하고 대상 애플리케이션에 텍스트 입력

## ✨ 기능

### 핵심 기능

| 기능 | 설명 |
|---------|-------------|
| 🎤 **지능형 음성 입력** | 눌러서 말하기, 놓아서 인식 및 입력 |
| 🧠 **로컬 ASR 인식** | Sherpa-ONNX 프레임워크 사용, 오프라인 작동 |
| 🌐 **다국어 인식** | 중국어, 영어, 일본어, 한국어, 광둥어 지원 |
| 📚 **개인 어휘** | AI 기반 수정 기능과 함께 사용자 정의 어휘 지원 |
| 📝 **기록 로그** | 입력 기록을 자동으로 저장, 쉽게 참조 가능 |
| 🔔 **트레이 통합** | 시스템 트레이로 최소화, 항상 접근 가능 |
| ⚡ **자동 시작** | 부팅 시 자동 시작 지원 |
| ⌨️ **글로벌 단축키** | 사용자 정의 가능한 단축키 |

### 지원되는 언어

WordFlow 의 음성 인식 모델은 다음 언어를 지원합니다:

| 언어 | 설명 |
|------|------|
| 중국어 (표준어) | 간체 중국어 음성 인식 |
| 영어 | 영어 음성 인식 |
| 일본어 | 일본어 음성 인식 |
| 한국어 | 한국어 음성 인식 |
| 광둥어 | 광둥어 음성 인식 |

> **참고**: 소프트웨어 UI 는 현재 중국어와 영어만 지원합니다.

### 단축키 설정

WordFlow 는 다양한 사용 습관에 맞는 단축키 구성을 지원합니다:

| 단축키 유형 | 설명 |
|-------------|-------------|
| 음성 입력 단축키 | 설정에서 사용자 정의 가능, 기본값은 ` 키 (그레이브 액센트) |

> **💡 팁**: 음성 입력 단축키로 한 손으로 쉽게 조작할 수 있는 키를 사용하는 것을 권장합니다.

### 모델 관리자

WordFlow 는 모델 관리 기능을 제공합니다:
- **원클릭 다운로드**: '다운로드'를 클릭하여 모델 직접 다운로드
- **자동 로드**: 다운로드 후 모델 로드 자동 시도
- **서비스 감지**: ASR 서비스가 실행되지 않은 경우 시작 프롬프트
- **상태 표시**: 실시간 모델 상태 (설치됨/미다운로드/로드됨)

## ❓ 자주 묻는 질문

### Q1: 첫 번째 실행 시 무엇을 해야 합니까?

첫 번째 실행 시 애플리케이션이 음성 인식 모델 파일 (약 150 MB) 다운로드를 안내합니다. 지침을 따르세요.

### Q2: 모델 다운로드가 실패하면 어떻게 해야 합니까?

프로그램 내 다운로드가 실패한 경우:
1. 네트워크 연결이 작동하는지 확인
2. 중국 사용자는 [Gitee 미러](https://gitee.com/yanlin-cheng/wordflow) 에서 다운로드 시도

### Q3: 모델을 다운로드한 후 왜 사용할 수 없습니까?

- 다운로드 후 모델 로드를 자동으로 시도
- ASR 서비스가 실행되지 않은 경우 시작 프롬프트
- 그래도 작동하지 않으면 애플리케이션을 재시작하거나 '서비스 연결' 버튼을 수동으로 클릭

### Q4: 인식이 부정확하면 어떻게 해야 합니까?

1. 마이크가 정상적으로 작동하는지 확인
2. 조용한 환경에서 사용
3. 명확하고 적절한 속도로 말하기
4. 개인 어휘에 자주 사용하는 어휘 추가

### Q5: 일부 애플리케이션에서 텍스트를 입력할 수 없는 이유는 무엇입니까?

일부 애플리케이션은 외부 입력을 제한할 수 있습니다. 다음을 시도하세요:
1. WordFlow 를 관리자로 실행
2. 대상 애플리케이션에서 텍스트 상자를 먼저 클릭하여 포커스 획득
3. 보안 소프트웨어가 차단하지 않는지 확인

### Q6: 단축키를 변경하려면 어떻게 해야 합니까?

1. 시스템 트레이 아이콘 오른쪽 클릭
2. '설정' 선택
3. '단축키 설정' 섹션에서 수정

### Q7: 부팅 시 자동 시작이 안 되는 이유는 무엇입니까?

1. '부팅 시 자동 시작' 옵션이 활성화되어 있는지 확인
2. 보안 소프트웨어가 자동 시작을 차단하지 않는지 확인
3. 작업 관리자의 '시작 프로그램' 탭에서 확인
