# WordFlow v1.0

WordFlow は、ローカル ASR 音声認識をサポートするインテリジェント音声入力ツールです。音声をリアルタイムでテキストに変換し、任意のアプリケーションに入力できます。

**v1.0 リリース** - .NET 8 WPF で構築され、安定したインテリジェントな音声入力体験を提供します。

## 🚀 クイックスタート

### インストール手順

1. **インストーラーをダウンロード**: `WordFlow_Setup.exe` をダウンロード
2. **インストーラーを実行**: ダブルクリックして指示に従う
3. **アプリケーションを起動**: インストール後、WordFlow を開始

### モデルのインストール

**方法 1：アプリ内でダウンロード（推奨）**
1. WordFlow を起動
2. 「モデル管理」ボタンをクリック
3. モデルの「ダウンロード」ボタンをクリック
4. ダウンロード完了後、モデルが自動的に読み込まれます

**方法 2：手動ダウンロード**
1. 上記のモデルファイルをダウンロード
2. `PythonASR/models/sensevoice-small-onnx/` ディレクトリに解凍
3. WordFlow を再起動

### 使用方法

1. **WordFlow を起動** - アプリケーションはシステムトレイに最小化されます
2. **押して話す** - 音声入力ホットキーを押して録音を開始
3. **離して入力** - ホットキーを離して認識し、ターゲットアプリケーションにテキストを入力

## ✨ 機能

### コア機能

| 機能 | 説明 |
|---------|-------------|
| 🎤 **インテリジェント音声入力** | 押して話す、離して認識して入力 |
| 🧠 **ローカル ASR 認識** | Sherpa-ONNX フレームワークを使用、オフラインで動作 |
| 📚 **個人用語彙** | AI 搭載の修正機能付きカスタム語彙をサポート |
| 📝 **履歴ログ** | 入力履歴を自動的に保存、簡単に参照可能 |
| 🔔 **トレイ統合** | システムトレイに最小化、常にアクセス可能 |
| ⚡ **自動スタート** | ブート時の自動スタートをサポート |
| ⌨️ **グローバルホットキー** | カスタマイズ可能なホットキー |
| 🔄 **自動モデル読み込み** | ダウンロード後、モデルを自動的に読み込み |

### ホットキー設定

WordFlow はさまざまな使用習慣に対応するホットキー設定をサポートしています：

| ホットキータイプ | 説明 |
|-------------|-------------|
| 音声入力ホットキー | 設定でカスタマイズ可能、デフォルトは ` キー（グレーブアクセント） |

> **💡 ヒント**: 音声入力ホットキーとして片手で簡単に操作できるキーの使用を推奨します。

### モデル管理

WordFlow はモデル管理機能を提供します：
- **ワンクリックダウンロード**: 「ダウンロード」をクリックしてモデルを直接ダウンロード
- **自動読み込み**: ダウンロード後、モデルの読み込みを自動的に試行
- **サービス検出**: ASR サービスが実行されていない場合、起動を促す
- **ステータス表示**: リアルタイムモデルステータス（インストール済み/未ダウンロード/読み込み済み）

## ❓ よくある質問

### Q1: 初回起動時に何をすればよいですか？

初回起動時、音声モデルファイル（約 150MB）のダウンロードが必要です。「モデル管理」→「ダウンロード」でアプリ内でダウンロードするか、モデルファイルを手動でダウンロードできます。

### Q2: モデルのダウンロードが失敗した場合、どうすればよいですか？

アプリ内でのダウンロードが失敗した場合：
1. ネットワーク接続が機能しているか確認
2. モデルファイルを手動でダウンロード（上記「モデルファイル」を参照）
3. `PythonASR/models/sensevoice-small-onnx/` ディレクトリに解凍

### Q3: モデルをダウンロードした後に使用できないのはなぜですか？

- ダウンロード後、モデルの読み込みを自動的に試行
- ASR サービスが実行されていない場合、起動を促す
- それでも動作しない場合、アプリケーションを再起動するか、「サービスに接続」ボタンを手動でクリック

### Q4: 認識が不正確な場合、どうすればよいですか？

1. マイクロフォンが正常に動作しているか確認
2. 静かな環境で使用
3. 明確に適切な速度で話す
4. 個人用語彙に頻繁に使用する語彙を追加

### Q5: 一部のアプリケーションでテキストを入力できないのはなぜですか？

一部のアプリケーションは外部入力を制限する場合があります。次を試してください：
1. WordFlow を管理者として実行
2. ターゲットアプリケーションでテキストボックスを先にクリックしてフォーカスを取得
3. セキュリティソフトウェアがブロックしていないか確認

### Q6: ホットキーを変更するにはどうすればよいですか？

1. システムトレイアイコンを右クリック
2. 「設定」を選択
3. 「ホットキー設定」セクションで変更

### Q7: ブート時に自動スタートしないのはなぜですか？

1. 「ブート時に自動スタート」オプションが有効になっているか確認
2. セキュリティソフトウェアが自動スタートをブロックしていないか確認
3. タスクマネージャーの「スタートアップ」タブで確認
