# WordFlow v2.0

**WordFlow** は、ローカル ASR 音声認識をサポートするスマート音声入力ツールです。音声をリアルタイムでテキストに変換し、あらゆるアプリケーションに入力できます。

**v2.0 完全リニューアル** - .NET 8 WPF で再構築され、より安定したインテリジェントな音声入力体験を提供します。

## 🚀 クイックスタート

### インストール手順

1. **インストーラーをダウンロード**: `WordFlow_Setup.exe` をダウンロード
2. **インストーラーを実行**: パッケージをダブルクリックし、指示に従ってインストール
   - .NET 8 ランタイムを自動的に検出・インストール
   - 音声認識モデルのダウンロードとインストールを選択（推奨）
3. **アプリを起動**: インストール完了後すぐに使用可能

### モデルのインストール

**推奨**: インストール中にモデルダウンロードオプションをチェックすると、インストーラーが自動的にモデルをダウンロードしてインストールします。

**代替方法**: インストール中にモデルがダウンロードされなかった場合：
1. WordFlow を起動
2. 「モデルマネージャー」ボタンをクリック
3. 目的のモデルの「ダウンロード」ボタンをクリック
4. ダウンロード完了後、自動的にモデルの読み込みを試みます

### 使用方法

1. **WordFlow を起動** - アプリはシステムトレイに最小化されます
2. **押して話す** - ホットキー（デフォルト：CapsLock）を押して録音開始
3. **離して入力** - ホットキーを離すと自動的にテキストを認識して入力

## ✨ 機能

### コア機能

| 機能 | 説明 |
|---------|-------------|
| 🎤 **スマート音声入力** | 押して話し、離して認識・入力 |
| 🧠 **ローカル ASR 認識** | アリババ DAMO 学院の Paraformer モデルを使用、オフラインで利用可能 |
| 📚 **個人用語彙** | カスタム語彙をサポート、AI スマート修正 |
| 📝 **履歴** | 入力履歴を自動保存、後で参照可能 |
| 🔔 **トレイ統合** | システムトレイに最小化、いつでもアクセス可能 |
| ⚡ **自動スタート** | ブート時の自動スタートアップをサポート |
| ⌨️ **グローバルホットキー** | カスタマイズ可能なホットキー |
| 🔄 **自動モデル読み込み** | ダウンロード後、自動的にモデルの読み込みを試みる |

### ホットキー設定

WordFlow は複数のホットキー設定をサポートしています：

| ホットキータイプ | デフォルト | 説明 |
|-------------|---------|-------------|
| 音声入力ホットキー | CapsLock | 押して話し、離して認識 |
| カスタムホットキー | 右 Alt | 設定で構成可能 |

> **💡 ヒント**: 音声入力ホットキーとして CapsLock を使用することを推奨します：
> - 中央に位置し、両手で簡単に操作可能
> - キー表面が大きく、誤触れが少ない
> - 押したときに明確な物理フィードバックがある

### モデル管理

v2.0 はモデル管理機能を追加：
- **ワンクリックダウンロード**: 「ダウンロード」ボタンを直接クリック、繰り返し選択不要
- **自動読み込み**: ダウンロード完了後、自動的にモデルの読み込みを試みる
- **サービス検出**: ASR サービスが実行されていない場合、ユーザーにプロンプトを表示
- **ステータス表示**: リアルタイムモデルステータス（インストール済み/未ダウンロード/読み込み済み）

## ❓ よくある質問

### Q1: 初回起動時に何がダウンロードされますか？

初回起動時、音声モデルファイル（約 206MB）を自動的にダウンロードします。ネットワークに接続したままにしてください。ダウンロード後は、再度ダウンロードせずに通常通り使用できます。

### Q2: モデルのダウンロードが失敗したらどうすればいいですか？

自動ダウンロードが失敗した場合：
1. ネットワーク接続が正常か確認
2. アプリケーションを再起動
3. モデルファイルを手動でダウンロード（上記「オプション 2」参照）
4. モデルマネージャーページで「更新」ボタンをクリック

### Q3: モデルをダウンロードした後、なぜ使用できないのですか？

v2.0 はこの問題を最適化しました：
- ダウンロード完了後、自動的にモデルの読み込みを試みる
- ASR サービスが実行されていない場合、ユーザーにプロンプトを表示
- それでも動作しない場合は、アプリを再起動するか、「サービスに接続」ボタンを手動でクリック

### Q4: 認識が不正確な場合はどうすればいいですか？

1. マイクが正常に動作しているか確認
2. 静かな環境で使用する
3. 明確に発音し、適度なスピードで話す
4. 個人用語彙に頻繁に使用する語彙を追加

### Q5: なぜ一部のアプリケーションでテキストを入力できないのですか？

一部のアプリケーションは外部入力を制限している可能性があります。次をお試しください：
1. WordFlow を管理者として実行
2. 対象アプリケーションでテキストボックスをクリックしてフォーカスを取得
3. セキュリティソフトウェアがブロックしていないか確認

### Q6: ホットキーをどのように変更しますか？

1. システムトレイアイコンを右クリック
2. 「設定」を選択
3. 「ホットキー設定」で変更

### Q7: ブート時に自動スタートしないのはなぜですか？

1. 「ブート時に自動スタート」がチェックされているか確認
2. セキュリティソフトウェアが自動スタートをブロックしていないか確認
3. タスクマネージャーの「スタートアップ」タブで確認
