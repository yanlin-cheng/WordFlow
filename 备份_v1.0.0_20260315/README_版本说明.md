# WordFlow v1.0.0 版本说明

## 版本信息
- **版本号**: v1.0.0
- **发布日期**: 2026 年 3 月 15 日

- **安装包大小**: 约 358MB

## 主要功能

### 1. 语音输入功能
- 按住热键（默认 `）录音
- 松开后自动识别并输入文字
- 支持中英文混合识别

### 2. 模型管理
- 支持在线下载语音识别模型
- 支持多模型切换
- 模型下载进度显示

### 3. 日志系统
- 自动记录运行日志
- 支持在应用内查看日志
- 日志文件位置：`%APPDATA%\WordFlow\logs\`

### 4. 设置功能
- 热键设置
- 语言设置（中文/英文）
- 开机自启动
- 检查更新

## 修复内容

### ASR 服务启动问题修复
**问题描述**: 虚拟机中 ASR 服务无法启动，显示路径错误

**原因**: 批处理文件使用相对路径，安装到 `C:\Program Files (x86)\` 后路径解析失败

**解决方案**: 
- 使用 `%~dp0` 获取批处理文件绝对路径
- 所有路径用引号包裹以处理空格
- 启动前切换到脚本目录

**修改文件**: `PythonASR/start_server.bat`

### 安装包优化
- 排除不需要的语言包（只保留中英文）
- 安装包大小从 374MB 减少到 358MB

## 系统要求

- **操作系统**: Windows 10/11 (64 位)
- **内存**: 至少 4GB RAM
- **磁盘空间**: 至少 500MB 可用空间
- **.NET 运行时**: 已内置，无需单独安装

## 安装说明

1. 解压 `publish_installer.rar`
2. 运行 `WordFlowSetup.iss` 创建安装包（可选）
3. 或直接使用 `publish_installer` 目录中的文件

## 使用说明

### 首次运行
1. 运行 `WordFlow.exe`
2. 如有提示，运行首次安装向导
3. 下载语音识别模型

### 日常使用
1. 在需要输入文字的界面
2. 按住 ` 键（波浪键）开始录音
3. 说话完毕后松开按键
4. 等待识别完成，文字自动输入

### 查看日志
1. 打开 WordFlow 设置
2. 点击"模型管理"
3. 点击"查看日志"按钮
