# WordFlow v1.0.0 构建说明

## 概述

本文档说明如何从源代码重新构建 WordFlow v1.0.0 安装包。

## 环境要求

### 必需软件
- **.NET 8 SDK** - https://dotnet.microsoft.com/download/dotnet/8.0
- **Git** - https://git-scm.com/
- **Inno Setup** (可选，用于创建安装程序) - https://jrsoftware.org/isdl.php

### 推荐配置
- Windows 10/11 (64 位)
- 至少 8GB RAM
- 至少 2GB 可用磁盘空间

## 构建步骤

### 步骤 1：获取源代码

```bash
# 克隆仓库
git clone https://github.com/wanddream/WordFlow.git
cd WordFlow

# 或切换到特定版本标签
git checkout v1.0.0
```

### 步骤 2：清理旧的构建产物

```bash
# 删除旧的构建目录
rm -rf bin/
rm -rf obj/
rm -rf publish_installer/
rm -rf Output/
```

### 步骤 3：发布项目

```bash
# 发布为自包含应用（包含 .NET 运行时）
dotnet publish WordFlow.csproj -c Release -r win-x64 --self-contained true -o publish_installer
```

**参数说明**：
- `-c Release`: 使用 Release 配置
- `-r win-x64`: 目标平台为 Windows 64 位
- `--self-contained true`: 包含 .NET 运行时，用户无需单独安装
- `-o publish_installer`: 输出目录

### 步骤 4：验证构建

构建完成后，检查以下目录结构：

```
publish_installer/
├── WordFlow.exe          # 主程序
├── Data/
│   └── models.json       # 模型配置文件
├── PythonASR/
│   ├── start_server.bat  # ASR 服务启动脚本
│   ├── asr_server.py     # ASR 服务 Python 脚本
│   ├── python/           # Python 运行时
│   └── models/           # 模型目录（初始为空）
├── zh-Hans/              # 中文语言包
└── en/                   # 英文语言包
```

### 步骤 5：创建安装程序（可选）

```bash
# 使用 Inno Setup 创建安装程序
# 打开 Installer/WordFlowSetup.iss 文件
# 在 Inno Setup 中编译
```

### 步骤 6：打包备份

```bash
# 压缩为 RAR 或 ZIP 格式
# Windows 资源管理器：右键 → 发送到 → 压缩文件夹
# 或使用 7-Zip、WinRAR 等工具
```

## 关键文件说明

### PythonASR/start_server.bat

此文件已修复路径问题，关键修改：

```batch
REM 使用 %~dp0 获取批处理文件所在目录的完整路径
set "SCRIPT_DIR=%~dp0"
set "PYTHON_EXE=%SCRIPT_DIR%python\python.exe"

REM 确保在正确的目录运行
cd /d "%SCRIPT_DIR%"
"%PYTHON_EXE%" asr_server.py --port 5000
```

### WordFlow.csproj

包含语言包优化配置，自动删除不需要的卫星程序集。

### Data/models.json

模型配置文件，包含可用模型列表和下载源信息。

## 常见问题

### Q1: 构建后安装包很大（>350MB）
**A**: 正常现象，因为包含了：
- .NET 8 运行时（约 100MB）
- Python 3.11 环境（约 100MB）
- Python 依赖包（约 50MB）

### Q2: 虚拟机中 ASR 服务无法启动
**A**: 确保使用修复后的 `start_server.bat`，关键点是：
- 使用 `%~dp0` 获取绝对路径
- 路径用引号包裹
- 启动前 `cd /d` 切换到脚本目录

### Q3: 语言包没有被排除
**A**: 运行 `dotnet clean` 后重新发布，确保 `CleanupSatelliteAssemblies` 目标被执行。
