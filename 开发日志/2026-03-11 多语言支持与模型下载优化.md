### 二、详细变更说明

#### 1. App.xaml.cs - UI 语言初始化
**新增功能：**
- 添加 `InitializeUICulture()` 方法
- 在应用程序启动时根据设置初始化 UI 语言
- 支持中文（简体）、英文、日文、韩文四种语言

**核心代码：**
```csharp
private void InitializeUICulture()
{
    var languageCode = _settingsService?.Settings.LanguageCode ?? "zh-CN";
    var cultureInfo = new CultureInfo(languageCode);
    
    // 设置当前线程的文化特性
    Thread.CurrentThread.CurrentCulture = cultureInfo;
    Thread.CurrentThread.CurrentUICulture = cultureInfo;
    
    // 设置 WPF 的默认语言
    FrameworkElement.LanguageProperty.OverrideMetadata(
        typeof(FrameworkElement),
        new FrameworkPropertyMetadata(
            XmlLanguage.GetLanguage(cultureInfo.IetfLanguageTag)));
}
```

#### 4. Services/ModelDownloadService.cs - 下载服务优化
**新增方法：**
- `DetectRegion()` - 检测用户所在区域（国内或国际）
- `GetOptimalDownloadSources()` - 获取最优下载源列表
- `DownloadFromSingleSourceAsync()` - 从单个下载源下载模型

**优化逻辑：**
1. 根据系统区域和语言自动检测用户位置
2. 国内用户优先使用 Gitee 下载源
3. 国际用户优先使用 GitHub 下载源
4. 支持自动故障切换，当前下载源失败时尝试下一个

### 四、技术细节

#### 1. 多语言资源文件结构
```
Resources/Strings/
├── Strings.resx           # 中文（简体）主资源文件
├── Strings.en.resx        # 英文资源文件
├── Strings.ja.resx        # 日文资源文件
├── Strings.ko.resx        # 韩文资源文件
└── Strings.Designer.cs    # 自动生成的代码访问类
```

#### 2. 区域检测逻辑
```
系统区域 (RegionInfo.CurrentRegion.Name)
    ├── CN 或 ZH → 返回 "cn"（国内）
    └── 其他 → 检查系统语言
            ├── zh- 开头且不含 TW/HK → 返回 "cn"
            └── 其他 → 返回 "global"（国际）
```

#### 3. 下载源优先级
```
国内用户：Gitee (priority:1) → GitHub (priority:2)
国际用户：GitHub (priority:1) → Gitee (priority:2)
```
