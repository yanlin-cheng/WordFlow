# WordFlow 项目交接文档

## 📋 项目概述

WordFlow 是一款智能语音输入工具，支持本地 ASR 语音识别，可将语音实时转换为文字并输入到任意应用程序中。

### 核心功能
- 🎤 智能语音输入 - 按住说话，松手自动识别上屏
- 🧠 本地 ASR 识别 - 使用 Paraformer 模型，离线可用
- 📚 个人词典 - 支持自定义词汇，AI 智能纠错
- 📝 历史记录 - 自动保存输入历史，方便查阅
- 🔔 托盘集成 - 最小化到系统托盘，随时调用

---

## 🏗️ 技术架构

### 技术栈
- **前端**: WPF (.NET 8.0)
- **语音识别**: Paraformer (ONNX Runtime)
- **音频处理**: NAudio
- **数据存储**: SQLite
- **Python 服务**: 本地 ASR 服务

## ⚠️ 注意事项

### 1. .NET 环境
- ✅ **无需安装** - 主程序已自包含 .NET 8.0 Runtime
- 开发时需要安装 .NET 8.0 SDK

### 2. Python 环境
- 支持三种方式（按优先级）：
  1. **嵌入 Python** - 打包在程序目录中（推荐）
  2. **虚拟环境** - 自动创建 venv
  3. **系统 Python** - 使用已安装的 Python 3.8+

### 3. 模型文件
- 模型文件较大（约 206MB），首次启动时自动下载
- 支持断点续传，下载中断后可恢复
- 模型保存在 `PythonASR/models/paraformer-zh/` 目录

### 4. 网络要求
- 首次启动需要联网下载模型
- 下载完成后可离线使用
