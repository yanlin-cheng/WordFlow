# WordFlow Python ASR 服务 - Sherpa-ONNX版本

轻量级离线语音识别，无需PyTorch，体积小、启动快。

## 特点

- **体积小**: 仅需sherpa-onnx (~50MB)，无需PyTorch
- **按需下载**: 首次启动选择模型下载，不浪费空间
- **多模型支持**: 支持Paraformer、SenseVoice等多种模型
- **热切换**: 运行时切换模型，无需重启

## 可用模型

| 模型 | 大小 | 语言 | 特点 |
|------|------|------|------|
| **paraformer-zh** | 200MB | 中文 | 中文识别率最高，推荐 |
| **sensevoice-small** | 100MB | 多语种 | 体积小，支持中英日韩粤 |
| **sensevoice-large** | 500MB | 多语种 | 多语种准确率最高 |

## 快速开始

### 1. 启动服务

```bash
cd PythonASR
start_server.bat
```

### 2. 首次使用

首次启动会提示选择模型：
```
1. Paraformer-中文  (200MB)  - 推荐中文用户
2. SenseVoice-Small (100MB)  - 多语种小模型  
3. SenseVoice-Large (500MB)  - 多语种大模型
```

选择后自动下载并启动。

### 3. 手动下载模型

```bash
python download_model.py paraformer-zh
```

## API接口

### 健康检查
```bash
GET http://127.0.0.1:5000/health
```

### 获取模型列表
```bash
GET http://127.0.0.1:5000/models
```

### 语音识别
```bash
POST http://127.0.0.1:5000/recognize
Content-Type: application/json

{
  "audio": "<base64-encoded-wav>",
  "sample_rate": 16000
}
```

### 切换模型
```bash
POST http://127.0.0.1:5000/load_model
Content-Type: application/json

{
  "model_id": "sensevoice-small"
}
```

## 常见问题

**Q: 下载速度慢？**
A: 模型托管在GitHub，国内可能需要代理。或使用国内镜像。

**Q: 内存不足？**
A: 选择sensevoice-small模型，仅需100MB。

**Q: 如何切换模型？**
A: 调用POST /load_model接口，或重启服务选择其他模型。

**Q: 支持热词吗？**
A: sherpa-onnx基础版不支持，需要热词请用paraformer-zh-spark版。
